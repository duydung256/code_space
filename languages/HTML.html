ウェブの基礎を学ぼう

HTMLとは？

HTMLは、ウェブページを作成するための標準マークアップ言語です。HTMLはHyperText Markup Languageの略で、ウェブページの構造を定義します。

HTMLはウェブページの骨組みを作り、テキスト、画像、動画、フォームなどのコンテンツを配置する方法を指定します。

HTMLの基本構造

<!DOCTYPE html>
<html>
<head>
<title>ページのタイトル</title>
</head>
<body>
<h1>これは見出しです</h1>
<p>これは段落です。</p>
</body>
</html>

HTMLの特徴

  • テキストベースのマークアップ言語
  • タグを使用して要素を定義
  • ブラウザによって解釈・表示される
  • CSSと組み合わせてスタイルを適用

必要なツール

  • テキストエディタ（VS Code, Sublime Textなど）
  • ウェブブラウザ（Chrome, Firefox, Safariなど）
  • 基本的なコンピュータスキル

Basic HTML Tags

HTML tags are used to define the structure of a webpage. Here are the most fundamental HTML tags with detailed explanations and examples:

Document Structure Tags

<!DOCTYPE html>

Declares that this is an HTML5 document. Must be the first line of the HTML file. 

<!DOCTYPE html>

<html></html>

The root element that contains all other HTML elements. The lang attribute specifies the language. 

<html lang="en">

</html>

<head></head>

Contains metadata about the document, including title, character encoding, styles, and scripts. 

<head>
<meta charset="UTF-8">
<title>Page Title</title>
</head>

<body></body>

Contains all the visible content of the webpage. 

<body>
<h1>Welcome to my website</h1>
<p>This is a paragraph.</p>
</body>

Text Tags

<h1> to <h6>

Headings from most important (h1) to least important (h6).

This is h1

This is h2

This is h3

This is h4

This is h5
This is h6
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>

<p></p>

Defines a paragraph of text. 

<p>This is a paragraph of text. It can contain multiple sentences and will automatically wrap to the next line when needed.</p>

<br>

Creates a line break. It is an empty element and doesn't need a closing tag. 

Line 1<br>Line 2

<strong> and <em>

Used for text emphasis. <strong> for strong importance, <em> for emphasized text.

This is strong text and this is emphasized text. 

<p>This is <strong>strong text</strong> and this is <em>emphasized text</em>.</p>

Links and Images

<a></a>

Creates a hyperlink to another webpage or resource.

Click here 
<a href="https://example.com">Visit Example.com</a>
<a href="mailto:email@example.com">Send email</a>
<a href="tel:+1234567890">Call us</a>

<img>

Embeds an image. Requires src attribute for the image source and alt attribute for alternative text. 

<img src="image.jpg" alt="Description of image" width="300" height="200">

Lists

<ul>, <ol>, and <li>

Create unordered (bulleted) and ordered (numbered) lists.

Unordered List:
  • Item 1
  • Item 2
  • Item 3
Ordered List:
  1. First item
  2. Second item
  3. Third item
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
<ol>
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ol>

HTML属性

HTML属性は、HTML要素に追加情報を提供します。属性は常に開始タグに指定され、通常は名前と値のペアで構成されます。

属性の例

<a href="https://www.example.com">リンクテキスト</a>
<img src="image.jpg" alt="画像の説明" width="500" height="300">

上記の例では、href、src、alt、width、height が属性です。

よく使われる属性

属性 説明 使用例
id 要素の一意の識別子 <div id="header">
class 要素のクラス名（CSSスタイリング用） <p class="highlight">
style インラインCSSスタイル <p style="color: blue;">
title 要素に関する追加情報（ツールチップ） <abbr title="World Wide Web">WWW</abbr>

HTMLフォーム

HTMLフォームは、ユーザーからデータを収集するために使用されます。フォームには、テキスト入力、チェックボックス、ラジオボタン、送信ボタンなどのさまざまな要素を含めることができます。

基本的なフォームの例

<form action="/submit-form" method="post">
<label for="name">名前:</label>
<input type="text" id="name" name="name" required>
<label for="email">メールアドレス:</label>
<input type="email" id="email" name="email" required>
<label for="message">メッセージ:</label>
<textarea id="message" name="message" rows="4" required></textarea>
<input type="submit" value="送信">
</form>

フォーム要素の例

主なフォーム要素

要素 説明
<form> フォームを定義
<input> 入力フィールドを定義（type属性で種類を指定）
<textarea> 複数行のテキスト入力フィールド
<label> 入力要素のラベル
<select> ドロップダウンリスト
<button> クリック可能なボタン
<fieldset> 関連する要素をグループ化
<legend> fieldsetのキャプション

セマンティックHTML

セマンティックHTMLとは、内容の意味を明確に伝えるタグを使用することです。これにより、ブラウザ、検索エンジン、開発者がコンテンツの意味を理解しやすくなります。

セマンティック要素の例

<header>
<h1>ウェブサイトのタイトル</h1>
<nav>
<ul>
<li><a href="#">ホーム</a></li>
<li><a href="#">サービス</a></li>
<li><a href="#">お問い合わせ</a></li>
</ul>
</nav>
</header>
<main>
<article>
<h2>記事のタイトル</h2>
<p>記事の内容...</p>
<section>
<h3>セクションの見出し</h3>
<p>セクションの内容...</p>
</section>
</article>
<aside>
<h3>関連情報</h3>
<p>サイドバーの内容...</p>
</aside>
</main>
<footer>
<p>著作権 © 2025 私のウェブサイト</p>
</footer>

主なセマンティック要素

<header>

ページまたはセクションのヘッダー部分を定義します。通常、ロゴ、見出し、ナビゲーションなどが含まれます。

<nav>

ナビゲーションリンクのセクションを定義します。サイト内の主要なナビゲーションに使用されます。

<main>

ドキュメントの主要なコンテンツを定義します。ページごとに一度だけ使用されるべきです。

<article>

独立したコンテンツを定義します。ブログ記事、ニュース記事、フォーラムの投稿などに適しています。

<section>

ドキュメント内の独立したセクションを定義します。通常、見出しを含みます。

<aside>

メインコンテンツに間接的に関連するコンテンツを定義します。サイドバーや広告などに使用されます。

<footer>

ページまたはセクションのフッター部分を定義します。著作権情報、連絡先、関連リンクなどが含まれます。

<figure> と <figcaption>

独立したコンテンツ（画像、図表など）とそのキャプションをグループ化します。
