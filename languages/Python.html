Pythonとは

Pythonは、シンプルで読みやすい構文を持つ高水準プログラミング言語です。1991年にグイド・ヴァン・ロッサムによって開発され、現在は世界中で広く使用されています。データ分析、機械学習、ウェブ開発、自動化など、さまざまな分野で活用されています。

Pythonの特徴

  • 読みやすく、書きやすい構文
  • 豊富なライブラリとフレームワーク
  • クロスプラットフォーム対応（Windows、Mac、Linux）
  • インタプリタ型言語（コンパイル不要）
  • 動的型付け
  • 自動メモリ管理
# Pythonの「Hello, World!」
print("こんにちは、世界！")

上記のコードは、Pythonでの最も基本的なプログラムの例です。print()関数を使用して、「こんにちは、世界！」というテキストをコンソールに出力します。

インストール方法

Pythonを使い始めるには、まずお使いのコンピュータにPythonをインストールする必要があります。以下に、主要なオペレーティングシステムでのインストール方法を説明します。

  1. Python公式サイトにアクセスし、最新バージョンをダウンロードします。
  2. ダウンロードしたインストーラーを実行します。
  3. 「Add Python to PATH」オプションにチェックを入れてください。
  4. 「Install Now」をクリックしてインストールを開始します。
  5. インストールが完了したら、コマンドプロンプトを開き、python --versionを実行してインストールを確認します。
  1. macOSには通常、Pythonがプリインストールされていますが、最新バージョンをインストールすることをお勧めします。
  2. Homebrewがインストールされていない場合は、まずHomebrewをインストールします。
  3. ターミナルを開き、brew install pythonを実行します。
  4. インストールが完了したら、python3 --versionを実行してインストールを確認します。
  1. 多くのLinuxディストリビューションには、Pythonがプリインストールされています。
  2. Ubuntuの場合、ターミナルを開き、sudo apt updateを実行してパッケージリストを更新します。
  3. 次に、sudo apt install python3 python3-pipを実行してPythonとpipをインストールします。
  4. インストールが完了したら、python3 --versionを実行してインストールを確認します。

変数と定数

Pythonでは、変数はデータを格納するためのコンテナです。変数名を定義し、値を代入することで使用します。Pythonは動的型付け言語であるため、変数の型を明示的に宣言する必要はありません。 

# 変数の宣言と代入
name = "田中太郎"  # 文字列
age = 30          # 整数
height = 175.5    # 浮動小数点数
is_student = True  # ブール値
# 変数の使用
print(f"{name}さんは{age}歳です。身長は{height}cmです。")
print(f"学生ですか？ {is_student}")

変数の命名規則

  • 変数名は文字（a-z、A-Z）、数字（0-9）、アンダースコア（_）で構成できます。
  • 変数名は数字で始めることはできません。
  • 変数名は大文字と小文字を区別します（name と Name は異なる変数です）。
  • 予約語（if、for、whileなど）は変数名として使用できません。

定数

Pythonには、他の言語のような真の定数はありませんが、慣例として、変更されるべきでない値を持つ変数は大文字で命名します。 

# 定数の慣例
PI = 3.14159
MAX_USERS = 100
DATABASE_URL = "postgresql://localhost/mydb"
# 使用例
circle_area = PI * radius * radius

複数の変数に同時に値を代入

Pythonでは、複数の変数に一度に値を代入することができます。 

# 複数の変数に値を代入
x, y, z = 1, 2, 3
# 値の交換
a = 5
b = 10
a, b = b, a  # aは10に、bは5になります

データ型

Pythonには、さまざまな組み込みデータ型があります。主要なデータ型は以下の通りです：

データ型 説明 例
整数 (int) 正または負の整数 42, -7, 0
浮動小数点数 (float) 小数点を含む数値 3.14, -0.001, 2.0
文字列 (str) テキストデータ "こんにちは", 'Python'
ブール値 (bool) 真または偽 True, False
リスト (list) 順序付けられた変更可能なコレクション [1, 2, 3], ["a", "b", "c"]

辞書 (dict) キーと値のペアのコレクション {"name": "田中", "age": 30}
セット (set) 重複のない要素のコレクション {1, 2, 3}, set([1, 2, 2, 3])

文字列の操作

Pythonでは、文字列を様々な方法で操作できます。

# 文字列の連結
first_name = "太郎"
last_name = "田中"
full_name = last_name + " " + first_name
print(full_name)  # 田中 太郎
# 文字列のフォーマット
age = 30

# 文字列のメソッド
text = "  Python Programming  "

print(text.replace("P", "J"))  # Jython Jrogramming

演算子

Pythonには、様々な演算子があります。演算子を使用して、変数や値に対して操作を行うことができます。

算術演算子

数値計算を行うための演算子です。

演算子 説明 例
+ 加算 5 + 3 = 8
- 減算 5 - 3 = 2
* 乗算 5 * 3 = 15
/ 除算（結果は常に浮動小数点数） 5 / 3 = 1.6666...
// 整数除算（小数点以下切り捨て） 5 // 3 = 1
% 剰余（除算の余り） 5 % 3 = 2
** べき乗 5 ** 3 = 125

比較演算子

値を比較するための演算子です。結果はブール値（TrueまたはFalse）になります。

演算子 説明 例
== 等しい x == y
!= 等しくない x != y
> より大きい x > y
< より小さい x < y
>= 以上 x >= y
<= 以下 x <= y

論理演算子

ブール値を組み合わせるための演算子です。

# 論理演算子
x = True
y = False
# and: 両方がTrueの場合にTrue
print(x and y)  # False
# or: 少なくとも1つがTrueの場合にTrue
print(x or y)   # True
# not: 真偽値を反転
print(not x)    # False

代入演算子

変数に値を代入するための演算子です。

# 基本的な代入
x = 10
# 複合代入演算子
x += 5   # x = x + 5 と同じ
print(x)  # 15
x -= 3   # x = x - 3 と同じ
print(x)  # 12
x *= 2   # x = x * 2 と同じ
print(x)  # 24
x /= 4   # x = x / 4 と同じ
print(x)  # 6.0
x //= 2  # x = x // 2 と同じ
print(x)  # 3.0
x %= 2   # x = x % 2 と同じ
print(x)  # 1.0
y = 2
y **= 3  # y = y ** 3 と同じ
print(y)  # 8

メンバーシップ演算子

シーケンス（文字列、リスト、タプルなど）内に要素が存在するかどうかを確認するための演算子です。 

# in: シーケンス内に要素が存在する場合にTrue
fruits = ["りんご", "バナナ", "オレンジ"]
print("バナナ" in fruits)  # True
print("ぶどう" in fruits)  # False
# not in: シーケンス内に要素が存在しない場合にTrue
print("ぶどう" not in fruits)  # True

条件分岐

条件分岐を使用すると、特定の条件に基づいてコードの実行を制御できます。Pythonでは、if、elif、elseステートメントを使用して条件分岐を実装します。

if文

最も基本的な条件分岐はif文です。条件がTrueの場合にのみ、指定されたコードブロックが実行されます。 

# 基本的なif文
age = 20
if age >= 18:
print("成人です。")

if-else文

if文にelseを追加すると、条件がFalseの場合に実行されるコードブロックを指定できます。 

# if-else文
age = 16
if age >= 18:
print("成人です。")
else:
print("未成年です。")

if-elif-else文

複数の条件を確認するには、elif（else if の略）を使用します。 

# if-elif-else文
score = 85
if score >= 90:
grade = "A"
elif score >= 80:
grade = "B"
elif score >= 70:
grade = "C"
elif score >= 60:
grade = "D"
else:
grade = "F"
print(f"あなたの成績は{grade}です。")

ネストされた条件分岐

条件分岐は、他の条件分岐の内部にネストすることもできます。 

# ネストされた条件分岐
age = 25
has_license = True
if age >= 18:
print("成人です。")
if has_license:
print("運転できます。")
else:
print("運転免許を取得してください。")
else:
print("未成年です。運転できません。")

条件式（三項演算子）

Pythonでは、簡単な条件分岐を1行で書くための条件式（三項演算子）があります。 

# 条件式（三項演算子）
age = 20
status = "成人" if age >= 18 else "未成年"
print(status)  # 成人

複数の条件の組み合わせ

論理演算子（and、or、not）を使用して、複数の条件を組み合わせることができます。 

# 複数の条件の組み合わせ
age = 25
income = 300000
if age >= 18 and income >= 200000:
print("ローン申請が可能です。")
else:
print("ローン申請の条件を満たしていません。")
# or演算子の使用
if age < 18 or income < 200000:
print("ローン申請の条件を満たしていません。")
else:
print("ローン申請が可能です。")
# not演算子の使用
is_holiday = False
if not is_holiday:
print("今日は仕事日です。")

ループ

ループを使用すると、コードブロックを繰り返し実行できます。Pythonには、主にforループとwhileループの2種類があります。

forループ

forループは、シーケンス（リスト、タプル、文字列など）の各要素に対して繰り返し処理を行うために使用されます。 

# リストの各要素に対するforループ
fruits = ["りんご", "バナナ", "オレンジ"]
for fruit in fruits:
print(fruit)
# 文字列の各文字に対するforループ
for char in "Python":
print(char)
# range()関数を使用したforループ
for i in range(5):  # 0から4まで
print(i)
# 開始値と終了値を指定したrange()
for i in range(2, 8):  # 2から7まで
print(i)
# ステップ値を指定したrange()
for i in range(1, 10, 2):  # 1, 3, 5, 7, 9
print(i)

whileループ

whileループは、指定された条件がTrueである限り、コードブロックを繰り返し実行します。 

# 基本的なwhileループ
count = 0
while count < 5:
print(count)
count += 1
# ユーザー入力を使用したwhileループ
password = ""
while password != "secret":
password = input("パスワードを入力してください: ")
if password != "secret":
print("パスワードが違います。もう一度試してください。")
print("ログインに成功しました！")

ループ制御ステートメント

Pythonには、ループの実行を制御するための特別なステートメントがあります。 

# break: ループを終了する
for i in range(10):
if i == 5:
break
print(i)  # 0, 1, 2, 3, 4
# continue: 現在の繰り返しをスキップして次の繰り返しに進む
for i in range(10):
if i % 2 == 0:  # 偶数の場合
continue
print(i)  # 1, 3, 5, 7, 9
# else句: ループが正常に終了した場合（breakで終了しなかった場合）に実行される
for i in range(5):
print(i)
else:
print("ループが正常に終了しました。")
# breakでループが終了した場合、else句は実行されない
for i in range(5):
if i == 3:
break
print(i)
else:
print("この文は表示されません。")

ネストされたループ

ループは他のループの内部にネストすることができます。 

# ネストされたループ
for i in range(1, 4):
for j in range(1, 4):
print(f"{i} x {j} = {i * j}")
print("-----")
# 九九の表
print("九九の表:")
for i in range(1, 10):
for j in range(1, 10):
print(f"{i} x {j} = {i * j:2d}", end="  ")
print()

関数の定義と呼び出し

関数は、特定のタスクを実行するためのコードブロックです。関数を使用することで、コードの再利用性が高まり、プログラムの構造が整理されます。

関数の定義

Pythonでは、defキーワードを使用して関数を定義します。 

# 基本的な関数の定義
def greet():
print("こんにちは！")
# 関数の呼び出し
greet()  # こんにちは！

戻り値を持つ関数

returnステートメントを使用して、関数から値を返すことができます。 

# 戻り値を持つ関数
def add(a, b):
return a + b
# 関数の呼び出しと戻り値の使用
result = add(5, 3)
print(result)  # 8
# 複数の値を返す関数
def get_min_max(numbers):
return min(numbers), max(numbers)
# 複数の戻り値を受け取る
min_val, max_val = get_min_max([1, 5, 3, 9, 2])
print(f"最小値: {min_val}, 最大値: {max_val}")  # 最小値: 1, 最大値: 9

ドキュメンテーション文字列（docstring）

関数の目的や使用方法を説明するために、ドキュメンテーション文字列（docstring）を追加することができます。 

# ドキュメンテーション文字列を持つ関数
def calculate_area(radius):
"""
円の面積を計算します。
引数:
radius (float): 円の半径
戻り値:
float: 円の面積
"""
return 3.14159 * radius * radius
# help関数を使用してdocstringを表示
help(calculate_area)

関数のスコープ

変数のスコープは、その変数にアクセスできるコードの範囲を定義します。 

# ローカル変数とグローバル変数
global_var = "グローバル変数"
def my_function():
local_var = "ローカル変数"
print(global_var)  # グローバル変数にアクセス可能
print(local_var)   # ローカル変数にアクセス可能
my_function()
print(global_var)      # グローバル変数にアクセス可能
# print(local_var)     # エラー: ローカル変数にはアクセスできない
# グローバル変数の変更
def modify_global():
global global_var
global_var = "変更されたグローバル変数"
modify_global()
print(global_var)  # 変更されたグローバル変数

パラメータと引数

関数は、パラメータを通じてデータを受け取ることができます。関数を呼び出す際に渡される値は引数と呼ばれます。

位置引数

最も基本的な引数の渡し方は、関数定義での順序に従って値を渡す方法です。 

# 位置引数
def greet(name, message):
print(f"{name}さん、{message}")
# 位置に基づいて引数を渡す
greet("田中", "おはようございます！")  # 田中さん、おはようございます！

キーワード引数

引数の名前を明示的に指定することで、順序に関係なく引数を渡すことができます。 

# キーワード引数
def greet(name, message):
print(f"{name}さん、{message}")
# 名前を指定して引数を渡す
greet(message="こんにちは！", name="佐藤")  # 佐藤さん、こんにちは！

デフォルト引数

パラメータにデフォルト値を設定することで、引数が省略された場合にデフォルト値が使用されます。 

# デフォルト引数
def greet(name, message="こんにちは！"):
print(f"{name}さん、{message}")
# messageを省略
greet("山田")  # 山田さん、こんにちは！
# messageを指定
greet("鈴木", "おやすみなさい！")  # 鈴木さん、おやすみなさい！

可変長引数

任意の数の引数を受け取るには、*args（位置引数）と**kwargs（キーワード引数）を使用します。 

# 可変長位置引数 (*args)
def sum_all(*numbers):
total = 0
for num in numbers:
total += num
return total
# 任意の数の引数を渡す
print(sum_all(1, 2))           # 3
print(sum_all(1, 2, 3, 4, 5))  # 15
# 可変長キーワード引数 (**kwargs)
def print_info(**info):
for key, value in info.items():
print(f"{key}: {value}")
# 任意の数のキーワード引数を渡す
print_info(name="高橋", age=30, city="東京")
# name: 高橋
# age: 30
# city: 東京

引数の順序

関数定義での引数の順序には規則があります。

# 引数の順序: 通常の引数 -> デフォルト引数 -> 可変長位置引数 -> 可変長キーワード引数
def example_function(arg1, arg2, arg3=None, *args, **kwargs):
print(f"arg1: {arg1}")
print(f"arg2: {arg2}")
print(f"arg3: {arg3}")
print(f"args: {args}")
print(f"kwargs: {kwargs}")
# 関数の呼び出し
example_function(1, 2, 3, 4, 5, 6, name="John", age=25)
# arg1: 1
# arg2: 2
# arg3: 3
# args: (4, 5, 6)
# kwargs: {'name': 'John', 'age': 25}

リスト

リストは、順序付けられた変更可能な要素のコレクションです。角括弧 [] で囲まれ、カンマで区切られた要素のリストとして表されます。

リストの作成

# 空のリスト
empty_list = []
# 要素を持つリスト
fruits = ["りんご", "バナナ", "オレンジ"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "こんにちは", 3.14, True]
# list()関数を使用したリストの作成
chars = list("Python")  # ['P', 'y', 't', 'h', 'o', 'n']

リストへのアクセス

# インデックスを使用した要素へのアクセス
fruits = ["りんご", "バナナ", "オレンジ", "ぶどう", "メロン"]
print(fruits[0])   # りんご（最初の要素）
print(fruits[2])   # オレンジ
print(fruits[-1])  # メロン（最後の要素）
print(fruits[-2])  # ぶどう（後ろから2番目の要素）
# スライスを使用した部分リストの取得
print(fruits[1:4])    # ['バナナ', 'オレンジ', 'ぶどう']
print(fruits[:3])     # ['りんご', 'バナナ', 'オレンジ']
print(fruits[2:])     # ['オレンジ', 'ぶどう', 'メロン']
print(fruits[::2])    # ['りんご', 'オレンジ', 'メロン']（1つおきに取得）
print(fruits[::-1])   # ['メロン', 'ぶどう', 'オレンジ', 'バナナ', 'りんご']（逆順）

リストの変更

# 要素の変更
fruits = ["りんご", "バナナ", "オレンジ"]
fruits[1] = "キウイ"
print(fruits)  # ['りんご', 'キウイ', 'オレンジ']
# 要素の追加
fruits.append("ぶどう")    # リストの末尾に追加
print(fruits)  # ['りんご', 'キウイ', 'オレンジ', 'ぶどう']
fruits.insert(1, "イチゴ")  # 指定した位置に挿入
print(fruits)  # ['りんご', 'イチゴ', 'キウイ', 'オレンジ', 'ぶどう']
# リストの結合
more_fruits = ["メロン", "パイナップル"]
fruits.extend(more_fruits)  # リストを拡張

タプルとリストの違い

タプルとリストの主な違いは、タプルが変更不可能（イミュータブル）であるのに対し、リストは変更可能（ミュータブル）であることです。

  • タプルは作成後に要素を変更、追加、削除できません。
  • タプルはハッシュ可能なため、辞書のキーとして使用できます（リストは使用できません）。
  • タプルはリストよりもメモリ効率が良く、アクセス速度が速いです。
  • タプルは関連するデータをグループ化するのに適しています（例：座標、名前と年齢のペアなど）。

プロパティ

プロパティを使用すると、メソッドをあたかも属性のように扱うことができます。これにより、属性へのアクセスを制御できます。 

class Circle:
def __init__(self, radius):
self._radius = radius  # プライベート属性の慣例

@property

return self._radius

raise ValueError("半径は正の数でなければなりません")
self._radius = value

return self._radius * 2

継承は、既存のクラス（親クラスまたは基底クラス）の属性やメソッドを新しいクラス（子クラスまたは派生クラス）が継承する機能です。これにより、コードの再利用性が高まります。

return "ニャー！"

抽象基底クラス

抽象基底クラス（ABC）を使用すると、サブクラスが特定のメソッドを実装することを強制できます。 

from abc import ABC, abstractmethod
class Shape(ABC):
@abstractmethod
def area(self):
pass
@abstractmethod
def perimeter(self):
pass
class Rectangle(Shape):
def __init__(self, width, height):
self.width = width
self.height = height
def area(self):
return self.width * self.height
def perimeter(self):
return 2 * (self.width + self.height)
class Circle(Shape):
def __init__(self, radius):
self.radius = radius
def area(self):
return 3.14159 * self.radius ** 2
def perimeter(self):
return 2 * 3.14159 * self.radius
# オブジェクトの作成と使用
# shape = Shape()  # エラー: 抽象クラスはインスタンス化できない
rectangle = Rectangle(5, 3)
circle = Circle(4)
print(f"長方形の面積: {rectangle.area()}")        # 長方形の面積: 15
print(f"長方形の周囲の長さ: {rectangle.perimeter()}")  # 長方形の周囲の長さ: 16
print(f"円の面積: {circle.area()}")              # 円の面積: 50.26544
print(f"円の周囲の長さ: {circle.perimeter()}")      # 円の周囲の長さ: 25.13272
