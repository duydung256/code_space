ウェブデザインの基礎を学ぼう

CSSとは？

CSSは、Cascading Style Sheetsの略で、HTMLで作成されたウェブページのデザインやレイアウトを指定するための言語です。CSSを使うことで、テキストの色やサイズ、背景色、レイアウト、アニメーションなど、ウェブページの見た目を自由にカスタマイズできます。

CSSはHTMLと組み合わせて使用され、「コンテンツ（HTML）」と「デザイン（CSS）」を分離することで、効率的なウェブ開発が可能になります。

CSSの基本構造

セレクタ {
プロパティ: 値;
プロパティ: 値;
プロパティ: 値;
}

CSSの特徴

  • HTMLと組み合わせて使用
  • カスケーディング（継承と上書き）の仕組み
  • セレクタでHTML要素を指定
  • プロパティと値でスタイルを定義

CSSの適用方法

  • インラインスタイル（HTML要素に直接記述）
  • 内部スタイルシート（HTMLファイル内のstyleタグ）
  • 外部スタイルシート（別ファイルのCSSを読み込み）

CSS構文

CSS構文は、セレクタとスタイル宣言ブロックで構成されています。セレクタはスタイルを適用するHTML要素を指定し、宣言ブロックにはプロパティと値のペアが含まれます。

CSS構文の例

/* 基本的なセレクタとスタイル宣言 */
h1 {
color: blue;
font-size: 24px;
margin-bottom: 10px;
}
/* クラスセレクタ */
.highlight {
background-color: yellow;
padding: 5px;
}
/* IDセレクタ */
#header {
background-color: black;
color: white;
}

CSSの適用方法

1. インラインスタイル

HTML要素のstyle属性に直接CSSを記述します。

<p style="color: blue; font-size: 16px;">これは青色のテキストです。</p>

2. 内部スタイルシート

HTMLファイルのhead要素内にstyleタグを使用してCSSを記述します。

<head>
<style>
p {
color: blue;
font-size: 16px;
}
</style>
</head>

3. 外部スタイルシート

別ファイル（.css）にCSSを記述し、HTMLファイルからlinkタグで読み込みます。

<head>
<link rel="stylesheet" href="styles.css">
</head>

styles.cssファイル：

p {
color: blue;
font-size: 16px;
}

セレクタ

セレクタは、スタイルを適用するHTML要素を指定するためのパターンです。CSSには様々な種類のセレクタがあり、特定の要素や要素のグループを正確に選択できます。

基本的なセレクタ

要素セレクタ

HTMLタグ名を使って要素を選択します。

p {
color: blue;
}
h1 {
font-size: 24px;
}

クラスセレクタ

特定のclass属性を持つ要素を選択します。

.highlight {
background-color: yellow;
}
.btn {
padding: 10px;
border-radius: 5px;
}

IDセレクタ

特定のid属性を持つ要素を選択します。

#header {
background-color: black;
}
#footer {
margin-top: 20px;
}

結合セレクタ

子孫セレクタ

特定の要素の子孫である要素を選択します。

div p {
margin-left: 20px;
}

子セレクタ

特定の要素の直接の子要素を選択します。

ul > li {
list-style-type: square;
}

隣接兄弟セレクタ

特定の要素の直後にある兄弟要素を選択します。

h1 + p {
font-weight: bold;
}

疑似クラスと疑似要素

疑似クラス

要素の特定の状態を選択します。

a:hover {
text-decoration: underline;
}
input:focus {
border-color: blue;
}
li:first-child {
font-weight: bold;
}

疑似要素

要素の特定の部分を選択します。

p::first-line {
font-weight: bold;
}
p::before {
content: "▶ ";
}
p::after {
content: " ◀";
}

ボックスモデル

CSSのボックスモデルは、すべてのHTML要素が矩形のボックスとして扱われる概念です。ボックスモデルは、コンテンツ、パディング、ボーダー、マージンの4つの部分で構成されています。

ボックスモデルの構成要素

  • コンテンツ (Content): テキストや画像など、要素の実際の内容が表示される領域
  • パディング (Padding): コンテンツの周りの透明な領域
  • ボーダー (Border): パディングの外側にある境界線
  • マージン (Margin): ボーダーの外側にある透明な領域で、他の要素との間隔を作る

マージン

ボックスモデルのプロパティ

width と height

要素のコンテンツ領域の幅と高さを設定します。

div {
width: 300px;
height: 200px;
}

padding

コンテンツ領域の周りの余白を設定します。

div {
padding: 20px;             /* 四方向すべて同じ値 */
padding: 10px 20px;        /* 上下 左右 */
padding: 10px 20px 15px;   /* 上 左右 下 */
padding: 10px 20px 15px 5px; /* 上 右 下 左 */
}

border

要素の境界線を設定します。

div {
border: 1px solid black;   /* 幅 スタイル 色 */
border-width: 2px;
border-style: dashed;
border-color: red;
}

margin

要素の外側の余白を設定します。

div {
margin: 20px;              /* 四方向すべて同じ値 */
margin: 10px 20px;         /* 上下 左右 */
margin: 10px 20px 15px;    /* 上 左右 下 */
margin: 10px 20px 15px 5px;  /* 上 右 下 左 */
}

box-sizing

ボックスモデルの計算方法を設定します。

div {
box-sizing: content-box;  /* デフォルト: width/heightはコンテンツ領域のみ */
box-sizing: border-box;   /* width/heightにパディングとボーダーを含む */
}

Flexbox

Flexbox（Flexible Box Layout）は、一次元のレイアウトモデルで、要素を行または列に配置し、利用可能なスペースを効率的に分配します。Flexboxを使用すると、要素の配置、整列、順序の変更などが簡単に行えます。

Flexboxの基本概念

Flexコンテナとアイテム

.container {
display: flex;
}

主軸と交差軸

  • 主軸 (Main Axis): flex-direction で定義される軸
  • 交差軸 (Cross Axis): 主軸に垂直な軸

レスポンシブデザイン

レスポンシブデザインは、様々な画面サイズやデバイスに対応するウェブデザインのアプローチです。CSSのメディアクエリを使用することで、デバイスの特性に応じてスタイルを変更できます。

メディアクエリの基本

メディアクエリは、特定の条件（画面サイズ、デバイスタイプなど）に基づいてCSSを適用するための条件式です。

@media screen and (max-width: 768px) {
/* 画面幅が768px以下の場合に適用されるスタイル */
.container {
flex-direction: column;
}
}
@media screen and (min-width: 769px) and (max-width: 1024px) {
/* 画面幅が769px以上1024px以下の場合に適用されるスタイル */
.container {
padding: 20px;
}
}

一般的なブレイクポイント

デバイス ブレイクポイント メディアクエリ例
スマートフォン（小） 320px - 480px @media (max-width: 480px) { ... }
スマートフォン（大） 481px - 767px @media (min-width: 481px) and (max-width: 767px) { ... }
タブレット 768px - 1024px @media (min-width: 768px) and (max-width: 1024px) { ... }
デスクトップ 1025px以上 @media (min-width: 1025px) { ... }

レスポンシブデザインの例

/* 基本スタイル（モバイルファースト） */
.container {
display: flex;
flex-direction: column;
padding: 10px;
}
.card {
width: 100%;
margin-bottom: 20px;
}
/* タブレット以上のスタイル */
@media (min-width: 768px) {

}

margin: 0 10px 20px;
}
}
/* デスクトップのスタイル */
@media (min-width: 1025px) {
.card {
width: calc(33.333% - 20px);
}
}

CSS変数

CSS変数（カスタムプロパティ）を使用すると、値を一度定義し、スタイルシート全体で再利用できます。これにより、コードの保守性が向上し、テーマの変更が容易になります。

CSS変数の基本

CSS変数は、--で始まる名前で定義し、var()関数で使用します。

:root {
--primary-color: #0000FF;
--secondary-color: #6699FF;
--text-color: #333333;
--font-size-base: 16px;
--spacing-unit: 8px;
}
.button {
background-color: var(--primary-color);
color: white;
padding: calc(var(--spacing-unit) * 2);
font-size: var(--font-size-base);
}
.button:hover {
background-color: var(--secondary-color);
}

CSS変数のスコープ

CSS変数は、定義された要素とその子孫に適用されます。:rootセレクタで定義すると、ドキュメント全体で使用できます。

:root {
--global-color: blue;
}
.container {
--local-color: red;
color: var(--global-color);  /* 青色 */
}
.container .child {
color: var(--local-color);   /* 赤色 */
}
.other-element {
color: var(--global-color);  /* 青色 */
color: var(--local-color);   /* エラー: 変数が定義されていない */
}

CSS変数の活用例

テーマの切り替え

CSS変数を使用すると、ダークモードなどのテーマ切り替えが簡単に実装できます。

--text-color: #333;
--border-color: #ddd;

--border-color: #555;
}

color: var(--text-color);

border: 1px solid var(--border-color);
}

レスポンシブデザインでの活用

メディアクエリと組み合わせて、画面サイズに応じて変数の値を変更できます。

:root {
--container-padding: 15px;
--heading-size: 24px;
}
@media (min-width: 768px) {
:root {
--container-padding: 30px;
--heading-size: 36px;
}
}
.container {
padding: var(--container-padding);
}
h1 {
font-size: var(--heading-size);
}

計算と組み合わせる

calc()関数と組み合わせて、動的な値を計算できます。

:root {
--spacing-unit: 8px;
}
.element {
padding: var(--spacing-unit);
margin-bottom: calc(var(--spacing-unit) * 2);
width: calc(100% - var(--spacing-unit) * 4);
}

CSS変数とJavaScriptの連携

JavaScriptを使用して、CSS変数の値を動的に変更できます。

// CSS変数の値を取得
const rootStyles = getComputedStyle(document.documentElement);
const primaryColor = rootStyles.getPropertyValue('--primary-color');
console.log(primaryColor);  // "#0000FF"
// CSS変数の値を設定
document.documentElement.style.setProperty('--primary-color', '#FF0000');
